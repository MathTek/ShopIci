# 📱 User Documentation - ShopIci

## Overview

ShopIci is a modern online marketplace that allows users to buy and sell local products. The application provides a smooth user experience with real-time chat features, cart management, and a favorites system.

## 🚀 Getting Started

### Registration and Login

1. **Create an account**:
   - Click "Sign Up" in the navigation bar
   - Fill out the form with your personal information
   - Validate your email if required

2. **Log in**:
   - Use your credentials (email/password)
   - "Remember me" option available

## 🛍️ Main Features

### Product Catalog

- **Browse products**: Home page with list of available products
- **Search and filters**: Quickly find what you're looking for
- **Product details**: Photos, descriptions, prices, seller information
- **Add to favorites**: Heart icon to save your favorite products

### Cart Management

- **Add products**: "Add to cart" button on each product page
- **Modify quantities**: Adjust quantities directly in the cart
- **Remove items**: Easily remove products
- **Automatic calculation**: Total updated in real-time

### Selling Products

1. **Create a listing**:
   - Go to "My Products" > "Add a product"
   - Upload photos
   - Fill in title, description, price, category

2. **Manage your listings**:
   - Edit or delete your products
   - View sales statistics

### Chat System

- **Contact sellers**: Chat button on each product page
- **Private conversations**: Secure messaging with buyers/sellers
- **Notifications**: Get notified of new messages

### User Profile

- **Personal information**: Manage your data
- **Purchase history**: View your past orders
- **Ratings**: User rating system

## 📱 User Interface

### Main Navigation

- **Home**: Product catalog
- **Products**: Advanced search
- **Cart**: Purchase management
- **Chat**: Messaging
- **Profile**: User settings
- **My Products**: Sales management
- **My Favorites**: Saved products

### Interactive Elements

- **Search bar**: Keyword search
- **Filters**: By category, price, location
- **Action buttons**: Add to cart, contact, favorites
- **Notifications**: New message indicators

## 🔒 Security and Privacy

### Data Protection

- **Secure authentication**: Login via email/password
- **Encrypted data**: Personal information protected
- **Secure payments**: Integration with reliable payment systems

### Community Rules

- **Mutual respect**: Courteous communication required
- **Compliant products**: Legal and authentic listings only
- **Reporting**: Button to report inappropriate content

## 📊 Technologies Used

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (Database, Authentication)
- **Deployment**: Vercel
- **Testing**: Vitest, Testing Library
